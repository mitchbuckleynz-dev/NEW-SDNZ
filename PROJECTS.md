# SDNZ Projects

Edit this file to update existing projects or add new ones.
Once you've made your changes, ask Antigravity to **sync this file into the website**.

---

## How to use

- **Edit** any project below by changing the values.
- **Add** a new project by copying the template at the bottom and filling it in.
- **Hide** a project without deleting it by setting `Published: false`.
- **Image**: either a path to a file in `/public/images/` (e.g. `/images/myproject.jpg`) or a full URL.
- `Slug` must be unique and URL-safe (lowercase, hyphens only, no spaces).
